Mi historia

Soy Lois Rodríguez Moimenta, un estudiante de un ciclo superior de Desarrollo de Aplicaiones Multiplataforma. Siempre me he fascinado por la tecnología. Me apasiona aprender nuevas tecnologías y mejorar mis habilidades constantemente.

Habilidades y Herramientas

Lenguajes

Me desenvuelvo principalmente en Java y Kotlin, con experiencia en desarrollo backend y aplicaciones móviles. También tengo conocimientos básicos en Python, JavaScript y C, lo que me permite adaptarme fácilmente a distintos entornos y desafíos de programación.

Front-End

He trabajado en la creación de interfaces web con HTML y CSS utilizando también el framework Bootstrap para lograr diseños responsivos y modernos. Me interesa seguir mejorando en la parte visual y de experiencia de usuario en mis proyectos personales.

Herramientas

Trabajo con sistemas operativos basados en Linux, y utilizo Git y GitHub para el control de versiones y la colaboración en proyectos, lo que me permite mantener un flujo de trabajo organizado y eficiente. Además, trabajo habitualmente con entornos de desarrollo como Visual Studio e IntelliJ IDEA, adaptándome según el tipo de proyecto y tecnología.

Bases de Datos

Tengo experiencia trabajando con bases de datos relacionales usando Microsoft SQL Server y lo básico de MongoDB. Además, utilizo herramientas como DBeaver para administrar y consultar bases de datos de forma eficiente. He realizado consultas SQL básicas y avanzadas para manipular y extraer datos según las necesidades de los proyectos.
